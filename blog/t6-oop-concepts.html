Blocks, Procs, and Lambdas

The right one for the right job

September 6, 2015

Blocs, Procs, and Lambdas, all three are referred to as closures in Computer Science. Most of the time, they can be used interchangeably, but they are not the same. Lambdas are objects.

Blocs are code passed to a method within '{}' or use do...end.

Procs are a block saved into a variable.

Lambdas are like procs.

...pretty simple? No

Here's a bit of differientation:

  • Blocs are for one time use, you con't reuse the code unless you type it all again (nooooooooooo)
  • Procs can be used again and again.
  • Lambdas check the number of arguments and will return the value to the method when it hits a 'return'.
